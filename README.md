# 🚀 200 Days of Machine Learning Challenge

> A personal challenge to build **deep, practical, and consistent Machine Learning skills** in 200 days.

---

## 🎯 Goal

This challenge is NOT about “finishing ML in 200 days.”

It IS about:

- ✅ Building strong ML fundamentals  
- ✅ Creating real-world projects  
- ✅ Practicing daily consistency  
- ✅ Learning both quality & quantity  
- ✅ Building a strong ML portfolio  

---

## 📌 Challenge Rules

✔️ No Zero Days  
✔️ Minimum 30–60 minutes daily  
✔️ Learn + Build + Reflect  
✔️ Track progress publicly  

---

## 🧠 Learning Formula

Each day includes at least ONE:

### 📘 Learn
- Theory
- Concepts
- Papers / Blogs
- Documentation

### 💻 Build
- Coding practice
- Mini projects
- Kaggle competitions
- Model implementations

### 📝 Reflect
- Notes
- Summaries
- Teaching others
- Posting progress
